## How to Take Screenshots

1. Open each website in a browser
2. Set viewport to ~1200px wide for consistency
3. Capture the hero section or main landing view
4. Save as PNG with the exact filename shown above
5. Optimize images for web (keep under 200KB if possible)

## Tips

- Use consistent screenshot dimensions
- Capture the most visually appealing part of each site
- Ensure screenshots showcase the key features
- Consider using browser dev tools to set consistent viewport size
